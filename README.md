# Smart Automated Plant Care System
#### Student Name: *Tiago de Gregori*   Student ID: *20119203*

Project Description

This project aims to develop a smart plant care system using a Raspberry Pi. The system will monitor soil moisture levels and environmental conditions to determine when a plant needs watering. When the soil becomes too dry, the system will automatically activate a small water pump to water the plant.

The Raspberry Pi will process the sensor data and send it over a network using HTTP or MQTT. The data will be displayed on a simple dashboard, allowing the user to monitor plant conditions in real time and receive alerts when action is taken.
